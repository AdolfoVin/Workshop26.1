## Rodando o projeto
1. Instale o **PostgreSQL** e o **pgAdmin**.
2. Crie um novo banco de dados no pgAdmin.
3. Abra a ferramenta de Query (`Query Tool`).
4. Copie o conteúdo do arquivo `modelo_fisico.sql` e execute-o.
