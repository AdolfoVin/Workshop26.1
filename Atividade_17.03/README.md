# Desafio: Banco de Dados de Jogos

Esta pasta do repositório contém a solução do desafio proposto no workshop da **Fábrica de Software (Semestre 2026.1)** em 17/03/2026. O projeto apresenta a estrutura de dados das principais *publishers* e seus respectivos jogos, incluindo atributos como nomes, datas de lançamento e vínculos de publicação.

##  Modelagem
O arquivo abaixo contém o script SQL para a criação do schema e inserção dos dados de teste:

* [Resposta_desafio.sql](./Resposta_desafio.sql)

## Como Rodar
1. Instale o **XAMPP** e o **MySQL Workbench**.
2. No Painel do XAMPP, inicie o módulo **MySQL**.
3. No MySQL Workbench, abra uma nova aba de Query.
4. Copie o conteúdo de `Resposta_desafio.sql` e execute-o (`Ctrl + Shift + Enter`).

## Tecnologias
* [XAMPP](https://www.apachefriends.org/pt_br/download.html)
* [MySQL Workbench](https://dev.mysql.com/downloads/workbench/)
